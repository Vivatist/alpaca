Bootstrap выберет соответствующую реализацию и зарегистрирует её в доменном слое автоматически.

## Использование bootstrap в тестах

```python
from core.application.bootstrap import build_worker_application
from settings import settings

bootstrap_app = build_worker_application(settings)
ingest = bootstrap_app.ingest_document
process_file = bootstrap_app.process_file_event
```

В pytest уже есть фикстуры `worker_app`, `ingest_pipeline`, `process_file_use_case`, поэтому можно просто запросить их в тестах.
